# Dataset Structure

Tempatkan dataset di folder `datasets/` dengan struktur berikut:

## RAVDESS
```
datasets/ravdess/
├── Actor_01/
│   ├── 03-01-01-01-01-01-01.wav
│   └── ...
├── Actor_02/
│   └── ...
└── Actor_24/
```
Download: [RAVDESS](https://zenodo.org/record/1188976)

## EmoDB
```
datasets/emodb/
├── 03a01Fa.wav
├── 03a01Nc.wav
└── ...
```
Download: [EmoDB](http://emodb.bilderbar.info/download/)

## SAVEE
```
datasets/savee/
├── DC_a01.wav
├── JE_h02.wav
└── ...
```
Download: [SAVEE](https://www.kaggle.com/datasets/ejlok1/surrey-audiovisual-expressed-emotion-savee)

## TESS
```
datasets/tess/
├── OAF_angry/
│   ├── OAF_angry_01.wav
│   └── ...
├── YAF_happy/
│   └── ...
└── ...
```
Download: [TESS](https://www.kaggle.com/datasets/ejlok1/toronto-emotional-speech-set-tess)

## CREMA-D
```
datasets/crema_d/
└── AudioWAV/
    ├── 1001_DFA_ANG_XX.wav
    ├── 1001_IEO_HAP_HI.wav
    └── ...
```
Download: [CREMA-D](https://github.com/CheyneyComputerScience/CREMA-D)

## EMOVO
```
datasets/emovo/
├── neu-m1-b1.wav
├── gio-f2-l3.wav
└── ...
```
Download: [EMOVO](https://zenodo.org/record/3464334)
